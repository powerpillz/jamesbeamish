Home-Lab VirtualBox

Creating a virtual networking environment in VirtualBox has proved to be a fantastic way in which to expand upon my existing networking skills and showcase them to potential employers. My current network includes an Active Directory with DNS and DHCP fallover protection with a 50/50 split scope between the primary and secondary DHCP servers. I also have a dedicated print/file server and a mail-exchange server.

MEMS Angle Sensor Arduino & 3D Printing/Design

The project that sparked my pursuit of a career in IT. I identified a need for a cheap, easy to use angle system for use on Underground Production Drills. My journey led me to design and 3d print a custom housing to mount an MPU6050 accelerometer/gyroscope, enabling it to connect to the I2C bus of an Arduino Micro-Controller. I began by modifying some open source quad-copter stablising code from github. Before long, I fell in love with coding and enrolled in university to complete a Bachelor of Computer Science.
